# chapter3 演算法
# 大 O 符號
* 也可稱「蘭道符號 (Landau symbol) 」
## 定義
* f 與 g 為由整數集合或實數集合對映至實數集合的函數。
* f(x) 是 O(g(x))，每當 x > k 時，存在常數 C 與 k 使 |f(x)| ≤ C|g(x)|
* 讀作 「f(x) 是 g(x) 的大 O」

## 常數 C 與 k
* 稱為「witness」
* 證明 f(x) 是 O(g(x)) 時需要

### 例子
* 證明 f(x)=x²+2x+1 是 O(x²)
* 解 : 當 x>1 時，x²>x 且 x²>1，因此 0 ≤ x²+2x+1 ≤ x²+2x²+x² = 4x²
* C = 4，k = 1
